MonteCarlo

    % This quicksort algorithm is extracted from Chapter 7, Introduction to Algorithms (3rd edition)
    \begin{algorithm}
    \caption{Quicksort}
    \begin{algorithmic}
    \PROCEDURE{Quicksort}{$A, p, r$}
        \IF{$p < r$}
            \STATE $q = $ \CALL{Partition}{$A, p, r$}
            \STATE \CALL{Quicksort}{$A, p, q - 1$}
            \STATE \CALL{Quicksort}{$A, q + 1, r$}
        \ENDIF
    \ENDPROCEDURE
    \PROCEDURE{Partition}{$A, p, r$}
        \STATE $x = A[r]$
        \STATE $i = p - 1$
        \FOR{$j = p$ \TO $r - 1$}
            \IF{$A[j] < x$}
                \STATE $i = i + 1$
                \STATE exchange
                $A[i]$ with $A[j]$
            \ENDIF
            \STATE exchange $A[i]$ with $A[r]$
        \ENDFOR
    \ENDPROCEDURE
    \end{algorithmic}
    \end{algorithm}

1. Introduction

The goal of MonteCarlo method is to calculate the heat transfert between two surface. MonteCarlo method use finite element and probabilist theory.

First, the MonteCarlo method calculates the view factor between two surface.

The MonteCarlo method consist to decompose the surfaces (emissive surface \(A_i\) and absortive surface \(A_j\)) in little finite element. Here the finite element is triangle.

Figure 1. Rays generate by \(A_i\) and intercept by \(A_j\)

We generate \(N_r\) random ray from \(N_s\) random source point, from emissive surface \(A_i\). Those rays touch the finite element of absorbent surface \(A_j\).

The local view factor between element \(e^i_k\) of \(A_i\) and element \(e^j_l\) of \(A_j\) is calculate by subsurface summation :

\[F_{e^i_k \rightarrow e^j_l} = \frac{n}{N} = \frac{n}{N_s N_r}\]

With :

  • \(N=N_s N_r\) the total number of rays generate by \(A_i\)

  • \(n\) the number of rays generate by \(A_i\) intercept by \(A_j\)

After calculate all local view factor, we can calculate the global view factor with :

\[F_{A_i \rightarrow A_j} = \frac{1}{A_i} \sum_{k=1}^{K}{ \left( Area(e^i_k) \sum_{l=1}^{L}{F_{e^i_k \rightarrow e^j_l}} \right) }\]

After calculate the view factor, it calculates the heat transfert between surfaces \(A_i\) and \(A_j\), with the formula :

\[q_{A_i \rightarrow A_j} = \sigma F_{A_i \rightarrow A_j} (T_j^4 - T_i^4)\]

With \(T_i\) the absolute temperature of \(A_i\) and \(T_j\) the absolute temperature of \(A_j\).

2. Some tool’s

2.1. Visibility of surface

Before calculate the view factor, it is interesting to know if the to surface are visible to each other.

For that, we use the oriented normal associated to element.

Set oriented surfaces element \(S_1\) and \(S_2\) (with oriented normal : \(\mathbf{n_1}\) and \(\mathbf{n_2}\)) :

\[S_1 \text{ and } S_2 \text{ are visible } \Longleftrightarrow \mathbf{n_1} \cdot \mathbf{n_2} < 0\]
Figure 2. Visibility between two surface

2.2. Distribution of random ray

For the emissive element, we generate some \(N_s\) source point and \(N_r\) ray from one source point.

For that we choose good supposition on matters : * homogeneous (the emission is the same for any position) * anisotropic (the emission is the same for any direction)

The distributions :

  • Source point \(O\) : \(\left\{\begin{eqnarray} x_O = x_0 + R \sqrt{\xi_2} cos(2\pi \xi_1) \\ y_O = x_0 + R \sqrt{\xi_2} sin(2\pi \xi_1) \end{eqnarray}\right.\) with \(\xi_1\) and \(\xi_2\) wich follow uniformely distribution in \([0,1]\) until in finite element.

  • Incident angle \(\phi\) of ray follow uniformely distribution in \([0,2\pi]\)

  • Azimuthal angle \(\theta\) of ray follow uniformely distribution in \([0,\pi]\)

For more detail, see the section Distribution of random rays.

2.3. Interception of ray by square

We have the conditon to intersection between right and square \(\mathcal{R}\) (of origin point \(P\) and directional vector \(\mathbf{e_1}\) and \(\mathbf{e_2}\)).

Figure 3. Interception of ray by square
\[\text{The ray } \mathcal{C} \text{ is intersept by } \mathcal{R} \Longleftrightarrow \left\{ \begin{eqnarray} \left( \mathbf{v} \wedge \mathbf{e_2} \right) \cdot \mathbf{e_1} \ne 0 \\ 0 \leq \frac{ \vec{PO} \cdot \left( \mathbf{e_2} \wedge \mathbf{v} \right)}{\left( \mathbf{e_2} \wedge \mathbf{v} \right) \cdot \mathbf{e_1} } \leq 1 \\ 0 \leq \frac{ \vec{PO} \cdot \left( \mathbf{e_1} \wedge \mathbf{v} \right)}{\left( \mathbf{e_1} \wedge \mathbf{v} \right) \cdot \mathbf{e_2} } \leq 1 \end{eqnarray} \right.\]

3. Algorithm

3.1. First Strategy

Here, for all element \(e^i_k\) of surface \(A_i\), we generate once a set of ray and for all element \(e^i_l\), we browse the set of ray to know wich ray is intercept.

\begin{algorithm}
    \caption{Calculation of View Factor}
    \begin{algorithmic}
    \PROCEDURE{Calculation of view factor($A_i$, $A_j$, $N_s$, $N_r$)}
    $A_i$ is surface decomposed by finite element $e^i_k$
    $A_j$ is surface decomposed by finite element $e^j_l$
    $N_s$ number of source point of one element
    $N_r$ number of ray from one source point

    $Area_i \leftarrow 0$ : area of surface $A_i$
    $sum_2 \leftarrow 0$

    \FOR $e^i_k$ in set of element of surface $A_i$ :

        \FOR $1 \leq i_s \leq N_s$ :
            generate a source point $s$
            \FOR $1 \leq i_r \leq N_r$:
                generate a ray $r$ from source point $s$
            \ENDFOR
        \ENDFOR

        $sum_1 \leftarrow 0$
        \FOR $e^j_l$ in set of element of surface $A_j$ :
            \IF $e^j_k$ visible from $e^i_l$ :
                $n \leftarrow 0$ : number of ray intercept
                \FOR ray $r$ in set of ray :
                    \IF $r$ is intercept by $e^j_k$:
                        $n \leftarrow n + 1$
                    \ENDIF
                \ENDFOR
                $F_{local} \leftarrow \frac{n}{N_s N_r}$
                $sum_1 \leftarrow sum_1 + F_{local}$
            \ENDIF
        \ENDFOR
        $Area_i \leftarrow Area_i + Area(e^i_k)$
        $sum_2 \leftarrow sum_2 + Area(e^i_k) sum_1$
    \ENDFOR
    $F_{global} \leftarrow \frac{1}{Area(A_i)} sum_2$
    \RETURN $F_{global}$
    \end{algorithmic}
\end{algorithm}

Complexity : \(O\left( K (L+1) N_s N_r \right) = O\left( K L N_s N_r \right)\), with \(K\) number of element of \(A_i\), \(L\) number of element of \(A_j\), \(N_s\) number of source point and \(N_r\) number of ray from one source point.

Disadvantage : demand more memories. (not really important)

3.2. Second Strategy

Here, for all element \(e^i_k\) of surface \(A_i\), we generate once a set of ray and for all ray, we browse the set of element \(e^j_l\) to know wich element intercept the ray.

Algorithm of calculation of view factor

\begin{algorithm}
    \caption{Calculation of View Factor}
    \begin{algorithmic}
    \PROCEDURE{Calculation of view factor($A_i$, $A_j$, $N_s$, $N_r$)}
    $A_i$ is surface decomposed by $K$ finite element $e^i_k$
    $A_j$ is surface decomposed by $L$ finite element $e^j_l$
    $N_s$ number of source point of one element
    $N_r$ number of ray from one source point

    $Area_i \leftarrow 0$ : area of surface $A_i$
    $sum_2 \leftarrow 0$

    \FOR $e^i_k$ in set of element of surface $A_i$ :
        $n \leftarrow \{n_l\}_{1\leq l \leq L}$ is a list of zeros
        \FOR $1 \leq i_s \leq N_s$ :
            generate a source point $s$
            \FOR $1 \leq i_r \leq N_r$:
                generate a ray $r$ from source point $s$
                $l \leftarrow 0$
                \WHILE $r$ isn't intercept by $e^j_l$ and $l \leq L$:
                    $l lefttarrow l+1$
                \ENDWHILE
                \IF $l \leq L$
                    $n_j \lefttarrow n_j+1$
                \ENDIF
        \ENDFOR
        $Area_i \leftarrow Area_i + Area(e^i_k)$
        ($F_{e^i_k \righarrow e^j_l} \leftarrow  \frac{n_j}{N_s N_r}$)
        $F_{e^i_k \righarrow A_j} \leftarrow \frac{sum(n_j)}{N_s N_r}$ or $= sum($F_{e^i_k \righarrow e^j_l} )$
        $sum_2 \leftarrow sum_2 + Area(e^i_k) F_{local} $
    \ENDFOR
    $F_{global} \leftarrow F_{global} + \frac{1}{Area(A_i)} sum_2$
    \RETURN $F_{global}$
    \end{algorithmic}
\end{algorithm}
The interception must contain the visibility between \(e^i_k\) and \(e^j_l\).

Complexity : worst case \(O\left( KN_sN_r \frac{L(L+1)}{1} \right)\), with \(K\) number of element of \(A_i\), \(L\) number of element of \(A_j\), \(N_s\) number of source point and \(N_r\) number of ray from one source point.

3.3. Third Strategy

Here, we take all couple of element \(e^i_l\) and \(e^j_k\) and we calculate the correspondant view factor.

\begin{algorithm}
    \caption{Calculation of View Factor}
    \begin{algorithmic}
    \PROCEDURE{Calculation of view factor($A_i$, $A_j$, $N_s$, $N_r$)}
    $A_i$ is surface decomposed by finite element $e^i_k$
    $A_j$ is surface decomposed by finite element $e^j_l$
    $N_s$ number of source point of one element
    $N_r$ number of ray from one source point

    $Area_i \leftarrow 0$ : area of surface $A_i$
    $sum_2 \leftarrow 0$

    \FOR $e^i_k$ in set of element of surface $A_i$ :
        $sum_1 \leftarrow 0$
        \FOR $e^j_l$ in set of element of surface $A_j$:
            \IF $e^i_k$ and $e^j_l$ are visible
                $n \leftarrow 0$
                \FOR $1 \leq i_s \leq N_s$ :
                    generate source point $s$
                    \FOR $1 \leq i_r \leq N_r$ :
                        generate ray $r$
                        \IF $r$ is intercept by $e^j_l$
                            $n \leftarrow n + 1$
                        \ENDIF
                    \ENDFOR
                \ENDFOR
                $F_{e^i_k \rightarrow e^j_l} \leftarrow \frac{n}{N_s N_r}$
            \ENDIF
            $sum_1 \leftarrow sum_1 + F_{e^i_k \rightarrow e^j_l}$
        \ENDFOR
        $Area_i \leftarrow Area_i + Area(e^i_k)$
        $sum_2 \leftarrow sum_2 + Area(e^i_k) sum_1
    \ENDFOR
    $F_{global} \leftarrow \frac{1}{Area(A_i)} sum_2$
    \RETURN $F_{global}$
    \end{algorithmic}
\end{algorithm}

Complexity : \(O\left( K L N_s N_r \right)\), with \(K\) number of element of \(A_i\), \(L\) number of element of \(A_j\), \(N_s\) number of source point and \(N_r\) number of ray from one source point.
