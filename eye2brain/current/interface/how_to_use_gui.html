Information on the graphical user interface

In order to run our first Eye2Brain application, we have to create a new instance that, in our case, corresponds to one patient.

Figure 1. Experiments web page of the MSO4SC Portal

Firstly, the user has to click on "Experiments" then "Instances" and finally "Add a new one" in order to reach the web page depicted in Fig. 1. Then the application has to be selected and, in particular for the Eye2Brain applications, the user has the possiblity to choose between:

  • Level 0: eye2brain-level-0

  • Level 1: eye2brain-level-1

  • Level 2: eye2brain-level-2

These applications are relevant in the clinical research and medical context. We provide however an advanced interface for developers and end-users who require a fine tuning of some advanced input parameters. The parameters marked advanced below are available only in the advanced interfaces associated with each level in the following applications

  • eye2brain-level-0-advanced

  • eye2brain-level-1-advanced

  • eye2brain-level-2-advanced

At this point all the inputs of the three different models are already filled with some default values in order to have always all the parameters specified. We present now the various input parameters that are available for the three different levels.
